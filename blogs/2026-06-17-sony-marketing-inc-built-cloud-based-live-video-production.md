Sony extended on-premises video production infrastructure into the cloud using AWS Elemental MediaConnect Router to manage 90+ live video sources for sports events. The hybrid architecture eliminated physical router bottlenecks, reduced equipment costs and setup complexity, and enabled multi-program production while maintaining broadcast quality through M2L-X cloud switchers.
